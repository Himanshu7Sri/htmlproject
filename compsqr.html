Completing Square

This is the another type of getting the solution of the quadratic equation , The process of completing the square is known as completing square
Case 1 - if any quadratic equation is given then for making it perfect square convert the coefficient of x term as ( from px to 2.a.b ) than add and substract b2 to that equation and make it a square . i'll give an example similar to this concept below . where equation are of the form x2 + px = 0 , so just do it by yourself .

let us take some example to understand it better --

Example - solve the quadratic eqution x2 + 8x = 0 using complete square method .
Solution - quadratic eqution x2 + 8x = 0
          x2 + 2.4x = 0
          x2 + 2.4.x + 42 - 42 = 0
          ( x + 4 )2 - 42 = 0
          ( x + 4 )2 = 42
          Taking underroot both side
          √( x + 4 )2 = √42
          ( x + 4 ) = ∓4
          x + 4 = -4 or x + 4 = 4
          x = -4 -4 or x = 4 - 4
          x = -8 or x = 0
