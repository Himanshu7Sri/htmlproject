Cross-Multiplication Method

till now we have seen many methods to solve the linear equation for some cases this methos is the best way to solve the linear equation . .
general form of the linear equations are -
a1x + b1y + c1 = 0
a2x + b2y + c2 = 0
Step 1 : multiply b2 in first equation
                a1b2x + b1b2y + c1b2 = 0
                b1a2x + b1b2 + b1c2 = 0

Step 2 : now substract the first equation from the second equation -
                (b1a2 - b2a1)x + (b1b2 - b2b1)y + (c1b2 - b1c2) = 0
                (b2a1 - b1a2)x = (c2b1 - b2c1)
                x = (c2b1 - b2c1) / (b2a1 - b1a2)

Step 3 : substituting the value of x in the first equation , we get -

                y = (c1a2 - c2a1) / (b2a1 - b1a2)

Example - solve the following linear equation using Cross-Multiplication method -
                5x + 3y = 35
                2x + 4y = 28
Solution -
Step 1 : multiply 4 in the first equation and 3 in the second equation
                4.5x + 4.3y - 4.35 =
                3.2x + 3.4y - 3.28 = 0

after multiplying equations are
                20x + 12y - 140 =
                6x + 12y - 84 = 0

Step 2 : substract the first equation from the second equation -
                (20 - 6)x + (12 - 12)y -(140 - 84) = 0
                14x - 56 = 0
                14x = 56
               => x = 4

Step 3 : now put the value of x in any of the two equation and get the value of y
                6x + 12y = 84
                6 x 4 + 12y = 84
                12y = 84 - 24
                12y = 60
               => y = 5
