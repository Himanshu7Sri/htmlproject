Introduction to the Polynomial

      (a + b)2 = a2 + 2ab + b2
      (a - b)2 = a2 - 2ab + b2
      (a + b)3 = a3 + 3ab(a + b) + b3
      (a - b)3 = a3 - 3ab(a - b) - b3
      (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ac
      a3 + b3 = (a + b)(a2 - ab + b2)
      a3 + b3 = (a + b)3 - 3ab(a + b)
      a3 - b3 = (a - b)(a2 + ab + b2)
      a3 - b3 = (a - b)3 + 3ab(a - b)
      a3 + b3 + c3 - 3abc = (a + b + c)(a2 + b2 + c2 - ab - bc - ac)
      Note - if ( a + b + c ) = 0 than ,
      a3 + b3 + c3 = 3abc

      (1 + 1/x)2 = 1 + 2/x + 1/x2
      (1 - 1/x)2 = 1 - 2/x + 1/x2
      (1 + 1/x)3 = 1 + 3/x(1 + 1/x) + 1/x3
      (1 - 1/x)3 = 1 - 3/x(1 - 1/x) - 1/x3
