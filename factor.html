Factorisation

since factorise of any polynomial tell us that remainder is 0 at that value of x . we factorise a quadratic polynomial ( like x2 + lx + m ) by splitting the middle term lx as ( ax + bx ) , such that a*b = m so , the result of the factorisation of the polynomial is -

x2 + lx + m = ( x + a )( x + b )

if the quadratic polynomial of the type ax2 + bx + c , where a ≠ 0 and a , b , c are constant . factorisation of these type of polynomial are of the form -

ax2 + bx + c = ( px + q )( rx + s )

now , ax2 + bx + c = ( prx2 + psx + qrx + qs )
   , ax2 + bx + c = ( prx2 + ( ps + qr )x + qs )

comparing both side , we get -
By comparing the coefficient of x2 , => a = pr
By comparing the coefficient of x , => b = ( ps + qr )
By comparing the ( x0 )constant term , => c = qs
these comparison shows us that b is the sum of two numbers ( i.e. ps and qr ) when we multiply those two number we get -
b = ( ps + qr )
=> multiplication of this number is
ps x qr = pr x qs
=> ac
At the end summery is this , we have to write b as the sum of two numbers whose product is ac .
