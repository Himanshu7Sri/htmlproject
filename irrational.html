Irrational Number

Decimal numbers are of the following forms -

(1) Terminating
(2) Non-Terminating but Repeating
(3) Non-Terminating and Non-Repeating

(1) Terminating - a decimal number in which, after a finite number of decimal places, all succeeding place values are (zero) 0 .
Example - 1/8 = 0.125


(2) Non-Terminating but Repeating - a decimal number in which, after a finite number of decimal places, all succeeding place values are not 0 but values are repeating continuously till infinity.
Example - 122/3 = 40.66666....


(3) Non-Terminating and Non-Repeating - a decimal number in which, after a finite number of decimal places, all succeeding place values are not 0 and the values are also not repeating till infinity.
Example - 3/7 = 0.428571428571428571....


Irrational Number -> a number is said to be irrational number , if it has non-terminating and non-repeating decimal representation.
Example - √3 , √5 , √7 is an irrational number
