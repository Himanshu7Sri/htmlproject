Area of Triangle

if we have given coordinate of three points that makes a triangle when we meet these three points so for calculating the area of triangle we have another formula that is -

Area of Triangle = 1 / 2 x base x height
Area of Triangle = 1 / 2 ( sum of parallel sides )( distance between them )

Area of Triangle = 1 / 2 [ x1( y2 - y3 ) + x2( y3 - y1 ) + x3( y1 - y2 ) ]

let us solve a question based on this concept -

Example 1 : Find the area of triangle whose coordinate is ( 0 , 0 ) , ( 2 , -4 ) and ( 5 , 10 ) .
Solution - using the above concept we can solve this question

Area of Triangle = 1 / 2 [ x1( y2 - y3 ) + x2( y3 - y1 ) + x3( y1 - y2 ) ]
                 = 1 / 2 [ 0( -4 - 10 ) + 2( 10 - 0 ) + 5( 0 - (-4) ) ]
                 = 1 / 2 [ 0( -14 ) + 2( 10 ) + 5( 4 ) ]
                 = 1 / 2 [ 0 + 20 + 20 ]
                 = 1 / 2 [ 40 ]
                 = 20
