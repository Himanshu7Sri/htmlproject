Division Algorithm of Polynomial

let p(x) be any polynomial of degree greater than or equal to one and let a be any real number . if p(x) is divided by the linear polynomial ( x - a ) then the remainder is p(a).


Dividend = ( Divisor x Quotient ) + Remainder

let us understand this concept by an example -

Example 1. Find the reamainder when x4 + x3 - 2x2 + x + 1 is divided by ( x - 1 ). Solution -
as according to the question x - 1 is completely divides x4 + x3 - 2x2 + x + 1 . so ,
=> x-1 = 0 => x = 1
put the value of x in the polynomial equation ...
=> p(1) = 14 + 13 - 2.12 + 1 + 1
=> p(1) = 1 + 1 - 2 + 1 + 1
=> p(1) = 2
